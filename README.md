# Personal Portfolio Website - Prodigy InfoTech Task 04

This project is a professional personal portfolio developed during my Web Development internship at Prodigy InfoTech. It showcases my skills, featured projects, and technical expertise in a modern, responsive layout.

## 🚀 Live Demo
You can view the live project here:

## ✨ Key Features
- **Modern UI:** Utilizes glassmorphism and neon styling for a sleek, contemporary look.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop screens.
- **Smooth Navigation:** Includes smooth-scroll effects and interactive UI components.
- **Project Gallery:** Displays recent work including a Weather App and Stopwatch.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for better accessibility.
- **CSS3:** Advanced styling including Flexbox, Grid, and animations.
- **JavaScript:** Logic for interactivity and dynamic elements.
